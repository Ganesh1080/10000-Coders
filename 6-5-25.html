Semantic tags

Document Structure

< html > - Root element of an HTML page
< head > - Contains metadata (title, links, etc.)
< body > - Main content of the HTML documentbr>

Layout Tags

< body > - Contains the content of the page
< nav> - Navigation links
< main > - Main content of the page (unique)
< section > - Group of related content
< article > - Self-contained content (e.g., blog post)
< aside > - Side content (e.g., sidebar, ads)
< footer > - Footer content (e.g., copyright, links)

Table Tags

< table > - Creates a table
< thead > - Table header section
< tbody > - Table body section
< tfoot > - Table footer section
< tr > - Table row
< th > - Header cell
< td > - Data cell
< caption > - Table title/description
